# レポジトリについて
これは、HTML,CSS,JavaScriptで構成されています。
## 使用方法
- ホームページでルールの確認などを行えます。
- 他のページへもアクセスすることができます。
## 技術スタック
HTML5
CSS3
JavaScript
## ライセンス
本プロジェクトはMITライセンスで公開されています。
## 貢献方法
1. リポジトリをフォーク
2. フィーチャーブランチを作成
3. プルリクエストを送信
4. 直接管理者にも連絡下さい。
